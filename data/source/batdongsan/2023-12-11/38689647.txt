   Chính chủ cần bán các lô đất thuộc Hòa Xuân, Cẩm Lệ, Đà Nẵng, như sau:
   <br/>
   1/ Lô đường Võ An Ninh ngày khu du lịch sinh thái ven sông Hòa Xuân thuộc phường Hòa Xuân, Câem Lệ, Đà Nẵng.
   <br/>
   Diện tích: 105m².
   <br/>
   Đất vuông vứt, đường 10,5m, vỉa hè rộng 5m.
   <br/>
   Gía 4.750 tỷ.
   <br/>
   2/ 2 lô đường Phạm Hữu Nghị, phía nam cầu Cẩm Lệ.
   <br/>
   Diện tích: 125m² và 100m².
   <br/>
   Đất vuông vức, sạch sẽ ko cống rãnh.
   <br/>
   Đường lớn 10.5m. Vỉa hè rộng 5m.
   <br/>
   3/ Lô đất đường Vũ Như Thạch, thuộc KDC Nam Hòa Xuân, Cẩm Lệ, Đà Nẵng.
   <br/>
   Diện tích 100m², đường rộng 7.5m.
   <br/>
   Hướng Nam, đối diện công viên.
   <br/>
   Gía: 2.65 tỷ.
   <br/>
   4/ lô đất đường Phan Khôi, thuộc KDC Nam Cầu Cẩm Lệ, Hòa Xuân, Đà Nẵng.
   <br/>
   Diện tích: 100m².
   <br/>
   Đường rộng 7.5m, vỉa hè 4m.
   <br/>
   Gía: 2.65 tỷ.
   <br/>
