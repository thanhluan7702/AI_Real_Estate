  <div class="re__section-body re__detail-content js__section-body js__pr-description js__tracking" trackingid="lead-phone-ldp" trackinglabel="loc=Rent-Listing Details-body,prid=38606867">
   Toà nhà văn phòng cho thuê nằm trên trục đường tài chính Nguyễn Văn Linh của thành phố Đà Nẵng.
   <br/>
   - Khu vực tập trung nhiều các toà văn phòng lớn, ngân hàng lớn.
   <br/>
   - Cư dân đông đúc, gần nhiều hàng quán cafe, quán ăn,..
   <br/>
   - Trung tâm thành phố, vị trí đẹp.
   <br/>
   <br/>
   Thông tin văn phòng + giá cả.
   <br/>
   + Không gian rộng rãi, đậu xe vỉa hè.
   <br/>
   + Tầng lửng: 70m².
   <br/>
   + Tầng 2: 105m².
   <br/>
   + Tầng 3: 105m².
   <br/>
   + Tầng 4: P1 - 60m².
   <br/>
   + Tầng 4: P2 - 45m².
   <br/>
   + Tầng 5: P1 - 55m².
   <br/>
   + Tầng 5: P2 - 50m².
   <br/>
   + Cho thuê dài hạn.
   <br/>
   + Giá thuê: 30 triệu/tháng/sàn - Hđ lâu dài - thanh toán 6 cọc 3.
   <br/>
   + Giá thuê nguyên toà: LH để được giá tốt hơn.
   <br/>
   <br/>
   - Ngoài ra còn các mặt bằng như sau:
   <br/>
   + Nguyên toà 4 tầng - diện tích 4.5*23.5m - tổng dtsd 368m².
   <br/>
   Giá thuê: 70tr/th - cọc 1 tháng thanh toán 6th/ lần.
   <br/>
   + Nguyên toà 3 tầng + 1 lửng - DTSD: 700m².
   <br/>
   Giá thuê: 120tr/th - Thanh toán 6 cọc 3.
   <br/>
   + Sàn văn phòng - Diện tích: 557m².
   <br/>
   Giá thuê: 220tr/th - Thanh toán 3 cọc 3 - HĐ: 3 năm (năm thứ 3 tăng không quá 10%).
   <br/>
   <br/>
   Professionalism. Honesty and Integrity.
   <br/>
   * Liên hệ ngay để được em tư vấn và dẫn trải nghiệm.
   <br/>
   * Sẵn sàng hỗ trợ, tư vấn 24/7.
   <br/>
   * Chuyên cho thuê, mua bán tất cả BDS khắp Đà Nẵng.
