   Đường Xô Viết Nghệ Tĩnh.
   <br/>
   Diện tích trống:
   <br/>
   75m² - tầng 4 - 160,000 vnd/m²/tháng (chưa VAT).
   <br/>
   75m² - tầng 5 - 180,000 vnd/m²/tháng (chưa VAT).
   <br/>
   Lợi thế: Hoạt động 24/7, dịch vụ quản lý tòa nhà chuyên nghiệp. Bãi đậu xe rộng rãi. Tuyến đường lớn, giao thông thoáng. Giờ làm việc thoải mái.
   <br/>
   <br/>
   Đường Võ Văn Tần.
   <br/>
   Diện tích trống: 65m² - tầng lửng.
   <br/>
   180,000 vnd/m²/tháng (chưa VAT).
   <br/>
   Lợi thế: Sẵn trần, sàn, thuận tiện setup. Có camera an ninh 24/24. Có máy phát điện công suất lớn, đảm bảo phục vụ nhu cầu làm việc liên tục tại đây. Giá thuê tốt.
   <br/>
   <br/>
   Đường 2/9.
   <br/>
   Diện tích trống:
   <br/>
   90,7m² - tầng 5.
   <br/>
   100m² - tầng 3.
   <br/>
   54m² - 90m² - tầng 1.
   <br/>
   172,500 vnd/m²/tháng (chưa VAT).
   <br/>
   Lợi thế: Tòa nhà mặt tiền, ngay cầu Tuyên Sơn, khu vực đông đúc. Có các trang thiết bị hiện đại bên trong văn phòng. Khu vực đậu đỗ xe rộng, thoải mái.
   <br/>
   <br/>
   Đường Đống Đa.
   <br/>
   Diện tích trống: 70m² - tầng 8.
   <br/>
   171,429 vnd/m²/tháng (chưa VAT).
   <br/>
   Lợi thế: Tòa nhà lớn, thiết kế đẹp, không gian đa năng. Chất lượng dịch vụ tối ưu, an ninh đảm bảo. Có tầng hầm giữ xe. Bàn giao nguyên sàn sẵn trần, sàn gạch men sạch sẽ.
   <br/>
   <br/>
   Đường Lê Đình Lý.
   <br/>
   Diện tích trống: 77,8m² - 77,2m² - tầng 3.
   <br/>
   275,000 vnd/m²/tháng (chưa VAT).
   <br/>
   Lợi thế: Tòa nhà đẹp, yên tĩnh chia ra làm 2 lô đôi. Đã có trần và sàn gạch men. Vị trí trung tâm, giao thông thoáng. Khu vực an ninh cao.
   <br/>
   <br/>
   Đường Hoàng Diệu.
   <br/>
   Diện tích trống:
   <br/>
   100m² - tầng 1 - 300,000 vnd/m²/tháng (chưa VAT).
   <br/>
   57m² - tầng 5 - 253,000 vnd/m²/tháng (chưa VAT).
   <br/>
   Lợi thế: Tòa nhà đẹp, có ngăn sẵn phòng tiện lợi. Đầy đủ hệ thống đèn điện, thang máy tốc độ cao. Khu vực dân cư văn minh, cách sân bay 2,4km.
   <br/>
   <br/>
   Đường Hùng Vương.
   <br/>
   Diện tích trống:
   <br/>
   96m² - tầng 1 - 575,000 vnd/m²/tháng (chưa VAT).
   <br/>
   94m² - tầng 9 - 184,000 vnd/m²/tháng (chưa VAT).
   <br/>
   Lợi thế: Đối diện chợ Cồn, bên cạnh TTTM Go. Khu vực đỗ xe rộng rãi. Dịch vụ quản lý tốt. Làm việc 24/24. Đã có trần, sàn cơ bản.
   <br/>
   <br/>
   Tỷ giá: 23.000vnd.
   <br/>
