   1. / Cho thuê lô đất trung tâm Hải Châu góc 3 mặt tiền gần cầu Trần Thị Lý.
   <br/>
   - Diện tích: 1000m² - ngang: 20m.
   <br/>
   - Vị trí đẹp, phù hợp nhà hàng, bar pub,...
   <br/>
   - Giá cho thuê: 120tr/tháng.
   <br/>
   Hợp đồng dài hạn 8 năm.
   <br/>
   2. / Cho thuê lô đất 2 mặt tiền đường Bạch Đằng trung tâm Hải Châu.
   <br/>
   - Diện tích: 1000m - ngang: 20m.
   <br/>
   - Vị trí đẹp phù hợp kinh doanh tất cả.
   <br/>
   - Giá cho thuê: 100tr/ tháng.
   <br/>
   - Hợp đồng lâu dài.
   <br/>
   3. / Cho thuê lô đất 2 mặt tiền đường Bạch Đằng trung tâm Hải Châu.
   <br/>
   - Diện tích: 2000m² - ngang: 40m.
   <br/>
   - Vị trí đẹp phù hợp kinh doanh tất cả.
   <br/>
   - Giá cho thuê: 200tr/ tháng.
   <br/>
   - Hợp đồng lâu dài.
   <br/>
   4. / Cho thuê lô đất 3 mặt tiền đường Bạch Đằng trung tâm Hải Châu.
   <br/>
   - Diện tích: 600m - ngang: 12m.
   <br/>
   - Vị trí đẹp phù hợp kinh doanh tất cả.
   <br/>
   - Giá cho thuê: 60tr/ tháng.
   <br/>
   - Hợp đồng lâu dài.
