   Mua đợt đầu mở bán mà giờ chuyển xuống nhà phố nên bán rẻ hơn thị trường 500 triệu.
   <br/>
   Căn hộ chung cư cao cấp dát vàng ven sông Hàn.
   <br/>
   - Nằm trên đường Như Nguyệt (Bạch Đằng nối dài) - trực diện sông Hàn Đà Nẵng.
   <br/>
   ----
   <br/>
   Thuộc cụm dự án Risemount.
   <br/>
   - DT: 76m².
   <br/>
   - Hướng BC Đông Nam - Tây Bắc.
   <br/>
   - 2 mặt tiền.
   <br/>
   + View sông Hàn + cầu Rồng.
   <br/>
   + View thành phố + biển Nguyễn Tất Thành.
   <br/>
   - 2 phòng ngủ.
   <br/>
   ----
   <br/>
   Tiện ích:
   <br/>
   - Hồ bơi vô cực, khu vui chơi, công viên, quầy bar, cafe trên mây...
   <br/>
   - Hầm để xe thông minh.
   <br/>
   ----
   <br/>
   Kết nối:
   <br/>
   - Nằm giữa lõi trung tâm tp. Đà Nẵng.
   <br/>
   - Mặt sông hàn.
   <br/>
   - Ngay điểm bắn pháo hoa.
   <br/>
   - Cách trung tâm trái bắp 1.5km.
   <br/>
   <br/>
   Sở hữu vị trí + view triệu đô + tòa nhà giác vàng.
   <br/>
   --------------------
   <br/>
   Vì nay kẹt nên đành bán rẻ hơn thị trường 500 triệu.
   <br/>
   Giá: 4,6 tỷ (thương lượng thêm).
