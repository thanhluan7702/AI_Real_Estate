   Thông tin mô tả

  <div class="re__section-body re__detail-content js__section-body js__pr-description js__tracking" trackingid="lead-phone-ldp" trackinglabel="loc=Sale-Listing Details-body,prid=38499226">
   Bán đất đường Bùi Trang Chước, phường Hoà Xuân, Đà Nẵng - Hình ảnh thực tế.
   <br/>
   - Diện tích: 100m².
   <br/>
   - Hướng: Đông Bắc.
   <br/>
   - Vị trí: Lên số nhà 324 Bùi Trang Chước, khu đông dân cư, nhà cửa cao tầng sang trọng.
   <br/>
   - Đường rộng 10,5m vỉa hè 5m, thuận tiện kinh doanh buôn bán hàng quán, mở nhà trẻ, trung tâm giáo dục,... Nhiều loại kinh kinh doanh.
   <br/>
   Giá bán chỉ 4,2 tỷ.
   <br/>
   - Sẵn sàng hỗ trợ vay 3 bên khách có nhu cầu, chủ ở Đà Nẵng.
   <br/>
   ----------------------------------------------------------------------------------
   <br/>
   Ngoài ra, trục đường Bùi Trang Chước còn nhiều lô khác, giá tuỳ thuộc vị trí:
   <br/>
   - B1.15 lô Tây nam 0x giá 4 tỷ.
   <br/>
   - B1.18 lô tây nam 5x giá 4,1 tỷ.
   <br/>
   - B1.24 lô đông bắc 9x giá 4.400 tỷ.
   <br/>
   - B1.37 lô đông bắc 3x giá 4.800 tỷ.
   <br/>
   - B1.42 lô kẹp cống - Gần góc Lê Quảng Chí - giá 4.600 tỷ.
   <br/>
   - B1.41 lô đông bắc giá 4.900 tỷ.
   <br/>
   - B1.42 lô 7x sạch giá 4,400 tỷ.
   <br/>
   Cặp.
   <br/>
   - B1.18 cặp tây nam, 200 m² giá 8 tỷ x.
   <br/>
   - B1.24 cặp đông bắc, 200 m² giá 8 tỷ x.
   <br/>
