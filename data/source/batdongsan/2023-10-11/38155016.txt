   <br/>
   Thông tin căn hộ Studio dự án Sun Cosmo Residence:
   <br/>
   <br/>
   Căn Studio căn số 10 và 11, diện tích 39.8m² thông thuỷ, tầng 3A.
   <br/>
   <br/>
   Giá thanh toán sớm 95%: 2 tỷ.
   <br/>
   <br/>
   Thanh toán tiến độ 30 tháng: 2.5 tỷ.
   <br/>
   <br/>
   Thanh toán tiến độ 12 tháng: 2.4 tỷ.
   <br/>
   ----------------------------
   <br/>
   <br/>
   Ngoài ra bên em còn các căn 1PN + 1, 2PN, 3PN.
   <br/>
   <br/>
   - Căn 01 - 2 - 03A - 7 - 8, 1PN + 1, view sông Hàn - Vòng quay mặt trời, ban công Đông Nam, giá từ 2.5 tỷ.
   <br/>
   <br/>
   - Căn 03 - 09 - 12, 2PN, view biển Mỹ Khê, ban công Đông Bắc, giá từ 4 tỷ.
   <br/>
   <br/>
   - Căn 06, 3PN, view trực diện sông Hàn - Thành phố, ban công Tây Nam giá từ 7 tỷ.
   <br/>
   ---------------------------------
   <br/>
   <br/>
   - Chính sách bán hàng linh hoạt.
   <br/>
   <br/>
   - Mỗi tháng thanh toán chỉ 2.5%.
   <br/>
   <br/>
   - Không vay, chiết khấu 6%.
   <br/>
   <br/>
   - Thanh toán sớm trong 12 tháng, chiết khấu 9%.
   <br/>
   <br/>
   - Thanh toán sớm trong tháng 9/2023, chiết khấu 12%.
   <br/>
   <br/>
   - Vay ngân hàng: Ân hạn nợ gốc và hỗ trợ lãi suất 0% đến 28/02/2026.
   <br/>
   <br/>
   - Booking ngay hôm nay được chiết khấu 2%.
